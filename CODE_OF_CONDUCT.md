# Community Code of Conduct

Mobazha aims to provide a professional, welcoming, and technically rigorous community.

## Expected behavior

- Be respectful and constructive, including when reviewing code or disagreeing on architecture.
- Focus criticism on ideas, behavior, and evidence rather than people.
- Welcome contributors with different backgrounds and levels of experience.
- Protect private information and follow responsible security-disclosure practices.
- Disclose relevant conflicts of interest when making project decisions.

## Unacceptable behavior

Harassment, discrimination, threats, sexualized attention, deliberate intimidation, doxxing, sustained disruption, and knowingly publishing another person's private information are not tolerated.

## Enforcement

Maintainers may edit or remove abusive content, close or lock discussions, reject contributions, or temporarily or permanently restrict participation. Enforcement decisions should be proportionate, documented privately, and avoid exposing reporters or affected people.

Report conduct concerns privately to the repository owners through the Mobazha GitHub organization. If the concern involves a security or privacy risk, use the private process in `SECURITY.md`. Do not post sensitive incident details in a public issue.
